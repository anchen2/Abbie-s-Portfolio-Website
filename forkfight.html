Fork Fight

UIUC Restaurant Ranking App

Project Overview

Fork Fight is a UIUC local food ranking app that provides students with a gamified way to find and sort their favorite restaurants.

The Problem

At th University of Illinois Urbana-Champaign, research revealed that while students often struggle deciding on where to eat around campus. Some students are uncertain about trying new spots on campus. Fork Fight aims to make these decisions easier while also introducing them to new places

Goals

  • Create a platform where students can find new restaurants.
  • Gamifying the question of “where should we eat” more fun and exciting.

"How Can We?" Statement

How can we ignite a passion for sustainable behaviors among UIUC students by developing a student-oriented platform that makes sustainability engaging, not only educating but also incentivizing and guiding behavioral change.

Ranking Game

  • Key features:
  • Select Mode: Show's all 3 voting modes, builds the ranking pages using an ELO algorithm (similar to chess.com).
  • Skip and Undo: Allows for user to undo mistakes and skip unknow restaurant pairing.s
  • Location Sharing: Helps the user locatate the restaurant and find the relative distance away.

UIUC Favorites

  • Key features:
  • Sort by: Users can sort by voted on Value, Aesthetic, or Speed of restaurants.
  • Maps: Helps the user to find out more and get directions to about top restaurant around campus.

My Favorites

  • Key features:
  • Share: Users can share to socail media thier top restaurants
  • Personalization: Top ranking list based only on the user's votes
